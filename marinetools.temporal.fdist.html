marinetools.temporal.fdist.copula module¶

class marinetools.temporal.fdist.copula.Copula(X, Y, family, *args)¶

Bases: object

This class estimates the copula parameter for the joint random variable. Copulas of Clayton, Frank y Gumbel are available. The base file comes from the ambhas 0.4.0 module of Sat Kumar Tomer, http://civil.iisc.ernet.in/~satkumar/, satkumartomer@gmail.com’

Some modifications are:

  1. Conditional copulas included for that families

  2. The package statistics is replaced by sklearn.neighbors.KernelDensity
